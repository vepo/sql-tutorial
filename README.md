# 📘 Bancos de Dados Relacionais & PostgreSQL – Tutorial

Bem-vindo a este tutorial para **Bancos de Dados Relacionais** e **PostgreSQL**. Este projeto fornece um ambiente completo baseado em Docker para explorar conceitos fundamentais de bancos de dados, desde álgebra relacional até modelagem entidade-relacionamento, utilizando conjuntos de dados do mundo real e exemplos práticos de SQL.

## 🚀 Início Rápido

### Pré-requisitos

* [Docker](https://docs.docker.com/desktop/setup/install/linux/) e [Docker Compose](https://docs.docker.com/compose/install/) instalados
* Pelo menos 2GB de RAM livre
* Familiaridade básica com linha de comando
